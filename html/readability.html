Tracklist Readability Settings

These settings determine when dividers are inserted into a release's track list in order to make them easier to read. You can show and hide dividers by clicking Show/Hide Dividers at the top-right area of the track list. Your show/hide preference will be saved each time you change it.

Note: After making changes here you must refresh your page on Discogs for changes to take effect.

How it works

There are some conditions that must be met before any dividers are inserted into the track list.

  • It wont insert dividers when Track Headings are already present on the release. These releases are usually readable on their own.
  • There must be sufficient tracks present before it does anything. The minimum number of tracks is determined by the Threshold setting. Once the Threshold has been exceeded, then the dividers will be inserted. For example, if you have a Threshold value of 8, any release with 9 or more tracks will have dividers inserted.
  • If there are vinyl tracks listed like: ABCDE (ie: there are no numbers after the side delineation — essentially one track per side), it won't insert dividers between those.
  • Note: There is no single standard when it comes to listing releases with multiple formats or CDs. Because of this, there is no guarantee that the dividers will be correctly inserted for every release.

Divider Size

This will set the size value for the dividers — either small, medium (default) or large.

Vinyl, Cassette, Multi-CD, Box Set Releases, etc …

Enabled:

This will insert dividers between sides/formats when there are multiple tracks per side (eg: A1, A2, B, C1, C2, or CD1-1, CD2-1, etc …) and the release exceeds the number of tracks listed here. If the release does not exceed this threshold, no dividers will be inserted.

Single CDs, Digital Releases, etc …

Enabled:

This will insert dividers when the release has only sequential numbers for track positions. This is most commonly seen in single CD releases or digital releases (eg: 1, 2, 3, 4, 5, 6, 7, etc …). Personally, I've found it helpful to set a high threshold so it's only active when viewing releases with a lot of tracks like this one.
